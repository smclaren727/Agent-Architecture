# External References

Collect outside inspiration here. Each reference should answer two questions:

1. What should we borrow?
2. What should we avoid?

Avoid dumping unlabeled screenshots into this folder. A short note makes each reference useful during implementation.

## Reference Categories

- `transparency-glass/` - translucent panels, blur, layered surfaces, macOS materials.
- `menus-popovers/` - compact menus, command surfaces, context menus, dropdowns.
- `dense-operator-uis/` - high-information operational tools that stay readable.
- `editor-reading/` - markdown/knowledge editors, reading-room layouts, properties/outline docks (Vault).
- `chat-composer/` - chat input, permission controls, model/runtime selectors.
- `color-material/` - palette, contrast, shadows, borders, surface treatment.
- `motion-interaction/` - subtle transitions, hover states, disclosure, focus.

## Reference Quality Bar

Prefer references that show:

- Real product UI over marketing shots.
- Actual menus, panels, tables, editors, and controls.
- Similar density to Vault or Overlay.
- Dark mode if that is the target direction.
- States with real text and controls, not empty decorative shells.
