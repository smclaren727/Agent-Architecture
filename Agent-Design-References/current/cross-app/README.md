# Cross-App Current Captures

Use this folder for side-by-side comparisons between Vault and Overlay.

## Useful Comparisons

- Main shell layout.
- Side panels and right docks.
- Menus and popovers.
- Buttons, toggles, segmented controls, and selects.
- Status badges and notices.
- Empty, loading, and error states.
- Typography scale and spacing rhythm.

## Questions

- Which visual elements should feel shared?
- Which elements should remain app-specific because Vault and Overlay have different jobs?
- Are colors carrying meaning consistently across both apps?
- Do transparency and blur help comprehension or just add decoration?
