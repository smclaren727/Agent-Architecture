# Current App Captures

Use this folder for screenshots of the apps as they operate now. These captures are the baseline for judging whether polish work is actually improving the product.

## Capture Checklist

- Vault shell with left navigation, editor, and right dock open.
- Vault Chat with an empty composer.
- Vault Chat with a prefilled convention prompt.
- Vault Conventions view with findings.
- Vault note editor with frontmatter and body content.
- Overlay shell home or workspace view.
- Overlay Automations view with Runner healthy.
- Overlay Automations view with a stale/missing Runner state if available.
- Overlay menus, popovers, confirmation states, and error notices.
- Any awkward clipped, crowded, low-contrast, or visually inconsistent state.

## Notes format (intentional deviation, 2026-07-06)

The 2026-07-06 baseline uses **consolidated `OBSERVATIONS.md` per folder** (`vault/`, `overlay/`,
`cross-app/`) instead of one paired note per screenshot. This is a deliberate choice, confirmed
with the owner, to keep the whole baseline readable in one place — each file still covers every
field in the per-capture template below (what feels good / rough / polish target / risk), just
aggregated. New captures may follow either form.
