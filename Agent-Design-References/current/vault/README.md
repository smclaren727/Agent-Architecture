# Vault Current Captures

Capture Vault states here.

## Priority States

- Notes lens with the editor active.
- Right dock Info tab.
- Right dock Chat tab, empty.
- Right dock Chat tab with a prefilled convention prompt.
- Chat with streamed/tool-bearing transcript content.
- Conventions lens with filters and findings.
- Structured editors for profiles, policies, tools, skills, workflows, standards, and memory facts where available.
- Loading, empty, unavailable, and error states.

## Watch For

- Composer height and clipping.
- Dense information that needs clearer grouping.
- Inconsistent panel backgrounds.
- Menus or overlays that feel too plain.
- Contrast problems in dark mode.
- Any visual treatment that makes Markdown/YAML ownership unclear.
